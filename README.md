# Tree Gallery

Welcome to **Tree Gallery**, a multi-page website showcasing different types of trees, including deciduous and evergreen varieties.

## About

This project has:

- A clean, responsive design that adapts to different screen sizes  
- Multiple pages: Home, Deciduous Trees, and Evergreen Trees  
- A navigation bar with active states to easily switch between pages  
- Image galleries with captions and descriptions for each tree  
- Custom fonts and icons for a polished look

## Technologies Used

- HTML  
- CSS (including media queries for responsiveness depending on screen size)  
- Google Fonts to fetch fonts
- Custom icons (svg files)

## File Structure

/index.html          - Home page
/deciduous.html      - Deciduous Trees gallery
/evergreen.html      - Evergreen Trees gallery
/style.css           - Main stylesheet
/images/             - Folder containing all tree images and icons
